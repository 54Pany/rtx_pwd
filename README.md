## 簡介 Introduction

RTX 是 Tencent 研發給企業內部使用的即時通訊軟體，但是其客戶端程式對於使用者密碼的保護非常薄弱。只需要在使用者的本地端執行本程式，就能從先前登入時已儲存的密碼或是執行中的 RTX process 記憶體裡尋回密碼明文。更多的細節可以參考 https://blog.debug.tw/rtx-and-tencent-suck/ 。
